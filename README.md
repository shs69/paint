### Добавлено:
1. История и возможность передвигаться по ней по кнопкам на экране
2. Горячие клавиши для истории (CTRL + Z - откатиться назад, CTRL + SHIFT + Z - вернуться, для macOS добавлены хоткеи через Cmd - Cmd + Z, Cmd + Y)
3. Добавлена возможность скачать изображение в формате png
4. Добавлена возможность загрузить изображение и отобразить его на холсте
